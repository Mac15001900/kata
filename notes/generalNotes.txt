To run this thing, inside the project folder, open 2 terminals.

In one of them run:
    ngrok http 3000 

In the other run either:
    npm run start
Or to use a debugger:
    node --inspect .\app.js

For the debugger, type chrome://inspect in Chrome, then select "Open dedicated DevTools for Node"
Local variables are not available to the debugger - either use breakpoints or attach them to "global" to make them available

To update commands, run:
    npm run register
Discord DMs update instantly; for server kick and re-invite the bot, or wait ~1 hour
